# Antigravity IDE Adapter

Antigravity carga `AGENTS.md` del proyecto como `user_rules` automáticamente.
Lee también `index.md`, el último `log.md`, y los context packs del proyecto.

Usa `roadmap/` para trabajo futuro y `roadmap/learnings/` para lecciones
reutilizables. Usa `roadmap-os repair` cuando una rotura de tooling o
metodología se descubra durante el trabajo.
